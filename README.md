# 🌱 Eco Tracker PWA

Прогрессивное веб-приложение для экологического образа жизни. Помогает сортировать отходы, находить пункты приёма вторсырья и отслеживать свой эко-след.

## 📐 Архитектура проекта

### Core Layer (Ядро)
- **Router** - SPA навигация между модулями
- **AuthService** - управление пользователем (заглушка)
- **DataService** - централизованный доступ к данным
- **StorageService** - работа с localStorage/IndexedDB
- **UIContainer** - базовая оболочка интерфейса

### Modules Layer (Модули)
1. **Waste Sort** - поиск и определение типа отходов
2. **Recycling Map** - карта пунктов приёма вторсырья
3. **Eco Tracker** - статистика и визуализация прогресса

### Data Layer
- Локальные заглушки данных для MVP
- В будущем: Firebase/Supabase
